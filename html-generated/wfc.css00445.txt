      <span class="salute">My Dear <span class="persName">Beck</span>
      </span>

      <p>I cannot make out if any one put up any money. And if there is any <a rel="tooltip" class="sic" title="likelihood​ ">likelyhood</a>
    ​ of any one going to put up— If no one is going to put up what is to be done. Shall we throw up the deal. I &amp; Mr <span class="persName">Salsbury</span> will not carry it alone. And if work cannot be commenced in Sep— we had better quit. I <a rel="tooltip" class="sic" title="can't​ ">cant</a>
    ​ for the life of me see what good was accomplished at last meeting—</p>

      <span class="closer">
         <span class="salute">
            <a rel="tooltip" class="sic" title="Hastily​ ">Hastly</a>
    ​ yours</span>

         <br/>Cody</span>

      <p>Has <span class="persName">Van Dreveldt</span> Been paid?</p>
