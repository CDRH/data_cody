      <span class="dateline">Olean. Aug 13th 95<a href="#n01" class="internal_link">1</a>
      </span>

      <span class="salute">My Dear <span class="persName">
            <a href="http://rosie.unl.edu/cody_archive/doc/beck.g">Beck</a>
         </span>
      </span>

      <p>I cannot make out if any one put up any money. And if there is any <a rel="tooltip" class="sic" title="likelihood​ &#xA;    ">likelyhood</a> of any one going to put up— If no one is going to put up what is to be done. Shall we throw up the deal. I &amp; Mr <span class="persName">
            <a href="http://rosie.unl.edu/cody_archive/doc/salsbury.n">Salsbury</a>
         </span> will not carry it alone. And if work cannot be commenced in Sep— we had better quit. I <a rel="tooltip" class="sic" title="can't​ &#xA;    ">cant</a> for the life of me see what good was accomplished at last meeting—</p>

      <span class="closer">
         <span class="salute">
            <a rel="tooltip" class="sic" title="Hastily​ &#xA;    ">Hastly</a> yours</span>

         <br/>Cody</span>

      <p>Has <span class="persName">
            <a href="http://rosie.unl.edu/cody_archive/doc/candreveldt.e">Van Dreveldt</a>
         </span> Been paid?</p>
